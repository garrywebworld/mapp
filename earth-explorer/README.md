# Earth Documentary Explorer

A premium static web application for exploring countries through a cinematic interactive Earth experience.

## Run Locally

Open `index.html` directly in a browser, or serve the folder with a simple local server:

```bash
python -m http.server 8766
```

Then visit:

```text
http://127.0.0.1:8766/
```

## GitHub Pages

This project is GitHub Pages friendly. Upload these files to the root of a repository:

- `index.html`
- `css/`
- `js/`
- `data/`
- `assets/`
- `images/`
- `videos/`
- `audio/`
- `fonts/`
- `icons/`

Then enable GitHub Pages from the repository settings.
